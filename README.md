URL Shortener API

FastAPI

PostgreSQL

Redis

Docker

\- создание короткой ссылки

\- redirect

\- обновление

\- статистика

\- кастомные alias

\- поиск

\- срок жизни ссылки

\- Redis кэширование

docker-compose up --build
